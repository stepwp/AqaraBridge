# Aqara 设备控制SDK支持设备清单

> 来源: https://opendoc.aqara.com/docs/SDK开发/iOS开发指南/设备控制/设备控制SDK支持设备清单.html

## 支持的设备品类

| 设备品类 | 是否支持 | 说明 |
| --- | --- | --- |
| 网关中控 | 支持 | 网关类产品 |
| 开关插座 | 支持 | 开关类、插座类产品 |
| 调光照明 | 支持 | 灯类产品 |
| 传感监测 | 支持 | 传感器类产品 |
| 门窗遮阳 | 支持 | 窗帘电机类产品 |
| 环境调节 | 支持 | 空调、地暖、新风等产品 |
| 无线遥控 | 支持 | 无线开关等产品 |
| 生活电器 | 支持 | 宠物喂食器、智能马桶等产品 |
| 摄像监控 | 不支持 | Aqara摄像机等产品 |
| 运动健康 | 不支持 | Aqara睡眠带等产品 |
| 智能门锁 | 不支持 | Aqara门锁等产品 |
